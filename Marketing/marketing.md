# 📢 Marketing Strategy – $100/month Budget
## 🎯 Goal
Maximize the growth of the Discord bot (users + servers) with a limited $100/month budget.
---
## 💸 Budget Allocation
### 1. Targeted Ads – $60
- Platforms:
 - Reddit Ads (Discord, bots, gaming-related subreddits)
 - Twitter / X Ads (targeting developers / gamers)
- Objective:
 - Drive traffic to the bot invite page
- Format:
 - Simple posts + visuals (GIFs showcasing the bot)
---
### 2. Sponsorships / Partnerships – $25
- Small to mid-sized Discord servers (1k–20k members)
- Approach:
 - Sponsored message
 - Let them test the bot on their server
- Objective:
 - Direct acquisition of active users
---
### 3. Bot Platforms – $10
- Promotion on:
 - Top.gg
 - Discord Bot List
- Paid visibility:
 - Sponsored votes
 - Premium listings (if available)
---
### 4. Tools / Assets – $5
- Visual creation:
 - Canva or other design tools
- Simple landing page hosting (optional)
---
## 📈 Organic Strategy (Free)
### 🔹 Content
- Post demos of your bot on:
 - TikTok
 - Twitter / X
 - YouTube Shorts
- Format:
 - Short videos showing useful features
### 🔹 Communities
- Actively engage in:
 - Reddit (r/DiscordBots, r/DiscordApp)
 - Developer Discord servers
- Avoid spamming → provide value
### 🔹 SEO / Presence
- Create a simple page (Notion or website)
- Optimize bot description with keywords
---
## 🧪 Experimentation
Each month:
- Test 1–2 new approaches
- Measure:
 - Cost per bot add
 - Number of active users
- Adjust budget based on results
---
## 📊 KPIs to Track
- Number of servers using the bot
- Number of active users
- Retention rate
- Cost per server acquisition
---
## 🚀 Long-Term Goal
Reach enough organic growth to reduce dependency on paid ads.
